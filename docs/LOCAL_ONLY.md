# Local-Only Work Rule

This repository is for local Tauri POC files only. The target runtime is Raspberry Pi CM5 / Debian Linux.

Do not deploy, copy, restart, or modify anything on:

- the CM5 totem
- the operator/server machine
- the senior reference machines
- the RTX AI machine
- the Windows server stack

Allowed actions here:

- create and edit local POC files
- document test procedures
- run local dependency install/build/dev commands on the CM5 test device only when explicitly approved

Before any real device deployment, make a separate checklist and get explicit approval.
