# Tauri CM5 POC

Purpose:
Evaluate whether Tauri can replace Electron for the Totem and Operator applications running on Raspberry Pi CM5 and Debian Linux industrial touch PCs.

Goals:
- Compare resource usage
- Test WebRTC
- Test camera
- Test microphone
- Test audio
- Test kiosk mode
- Test touchscreen
- Test ARM64 compatibility

Status:
Planning Phase
