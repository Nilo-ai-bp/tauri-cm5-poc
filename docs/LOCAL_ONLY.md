# Local Only

This repository is for local Tauri POC files only. The target runtime is Raspberry Pi CM5 / Debian Linux.

Do not use this repo to:

- deploy to the PRM totem
- deploy to the PRM server
- deploy to the PRM operator machines
- change Asterisk
- change RTX AI services
- modify senior reference machines

All existing PRM machines and services should be treated as external test dependencies only.
